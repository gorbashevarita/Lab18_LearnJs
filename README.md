#### 2. Подключение JavaScript к HTML
- способы подключения скрипта:
  - тег `<script>` в `<body>`;
  - тег `<script>` в `<head>` с атрибутом `defer`;

#### 3. Основы синтаксиса JavaScript
- объявление переменных:
  - `let` — изменяемая переменная;
  - `const` — константа;
  - `var` — устаревший способ (не рекомендуется);
- типы данных:
  - примитивные: `string`, `number`, `boolean`, `undefined`, `null`, `BigInt`, `Symbol`;
  - объектные: `object`, `array`, `function`, `Date`;
- особенности работы с `null` и `undefined`;
- понятие `NaN` и его особенности: `NaN === NaN` возвращает `false`.

#### 4. Преобразования типов
- явные: `Number()`, `String()`, `parseInt()` и др.;
- неявные (coercion) — особенности конкатенации и арифметических операций со строками (например: `"5" + 3 → "53"`, `"5" - 3 → 2`).

#### 5. Операторы сравнения
- нестрогое сравнение (`==`) — с приведением типов;
- строгое сравнение (`===`) — без приведения типов;
- сравнение объектов и массивов — по ссылке, а не по содержимому.

#### 6. Работа с консолью браузера
  - `console.log()`;
  - `console.clear()` — очистка консоли;

#### 7. Node.js
- назначение:
  - запуск JS вне браузера;
  - работа с файлами;
  - серверная разработка;
  - инструменты разработки (React, Webpack, npm);
- установка и проверка версии:
  - `node -v`;
  - `npm -v`;
- запуск скриптов: `node main.js`.

#### 10. Условные операторы и логика
- `if/else`, `else if`;
- логические операторы: `&&` (И), `!` (НЕ);
- тернарный оператор — для коротких условий;
- конструкция `switch/case` — с особенностью «проваливания» (fallthrough) без `break`.
